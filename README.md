# Prompt Library v1

## Objective

This project demonstrates how to create a reusable prompt template for customer support replies. Instead of writing a new prompt every time, the same template is reused by changing only the customer message.

---

## Use Case

Customer Support Replies

---

## Prompt Template Structure

- Role
- Context
- Task
- Format
- Constraints

---

## AI Tool Used

- ChatGPT

---

## Files Included

- README.md
- prompt-template.txt
- prompt-library.md

---

## Conclusion

This project demonstrates the importance of reusable prompt templates in prompt engineering. Using a consistent template improves efficiency, maintains response quality, and reduces the need to rewrite prompts from scratch.
